Простой, но удобный веб-генератор ретро-окон в стиле Windows XP и Windows 98. Создан специально для видеомонтажеров, стримеров и контент-мейкеров, которым нужны качественные оверлеи с прозрачным центром (хромакеем).
Больше не нужно искать PNG-картинки сомнительного качества в интернете или вручную вырезать центры окон в фотошопе. Генерируй идеальные рамки под любое разрешение прямо в браузере!
